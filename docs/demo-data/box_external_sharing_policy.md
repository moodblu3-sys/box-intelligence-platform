# Box外部共有運用ルール

対象顧客: 日新テクノロジー株式会社
管理部門: 情報システム部門
対象システム: Box

## 目的

取引先、協力会社、監査法人など社外関係者とBoxフォルダを共有する際の運用ルールを定める。外部共有は業務上必要な範囲に限定し、フォルダの機密区分、共有方法、取引先ドメインの許可状況を確認したうえで実施する。

## 外部共有の基本条件

Boxフォルダを外部ユーザーに共有できるのは、次の条件をすべて満たす場合に限る。

1. フォルダの機密区分が「公開可」または「社外共有可」である。
2. フォルダオーナーまたは共同所有者が共有を実施する。
3. 共有先メールアドレスが取引先の会社メールアドレスである。
4. 取引先ドメインが外部共有許可リストに登録されている。
5. 共有目的、共有期間、共有範囲が業務上必要な範囲に限定されている。

## 共有リンクの扱い

共有リンクは、機密区分が低い資料を一時的に共有する場合のみ利用できる。社外秘以上のフォルダでは、共有リンクによる外部共有は禁止する。必要な場合は、外部コラボレーターとして招待し、ユーザー単位で権限を付与する。

共有リンクを使う場合は、次の設定を確認する。

- リンクの対象範囲が「リンクを知っている全員」になっていないこと
- 有効期限が設定されていること
- 必要に応じてパスワードが設定されていること
- ダウンロード可否が業務目的に合っていること

## 外部コラボレーター招待

取引先に継続的にBoxフォルダへアクセスしてもらう場合は、外部コラボレーター招待を利用する。招待権限はフォルダオーナーまたは共同所有者が持つ。

招待時の確認事項:

- 招待先メールアドレスに誤りがないこと
- 付与する権限が最小権限であること
- 共有期間が決まっている場合は終了日を管理すること
- 退職者、担当変更者がフォルダオーナーのままになっていないこと

## 取引先ドメイン許可

取引先のメールドメインが外部共有許可リストにない場合、一般社員は直接招待できない。部門長承認を取得したうえで、情シス部門に取引先ドメイン許可申請を行う。

申請に必要な情報:

- 取引先会社名
- 許可したいメールドメイン
- 共有対象フォルダ
- 共有目的
- 共有予定期間
- 部門長承認の記録

## 問い合わせ時に確認する情報

外部ユーザーから「アクセスできない」と連絡があった場合、社員は次の情報を情シスへ伝える。

- 対象BoxフォルダのURL
- 招待したメールアドレス
- 取引先ドメイン
- フォルダの機密区分
- 共有リンクか外部コラボレーター招待か
- 表示されているエラーメッセージ

## 注意事項

Box AIや生成AIで外部共有ルールを確認する場合も、最終判断は本運用ルールに従う。権限外の共有、機密区分に反する共有、承認なしの取引先ドメイン追加は認めない。
