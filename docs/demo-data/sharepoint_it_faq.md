# 情シスFAQ: Box外部ユーザーがアクセスできない場合

対象顧客: 日新テクノロジー株式会社
掲載場所: SharePoint 情シス部門ポータル

## FAQ 1. 取引先がBoxフォルダにアクセスできないと言っている

まず、相手が招待されたメールアドレスでBoxにログインしているか確認する。個人メール、別会社メール、過去に使っていたBoxアカウントでログインしている場合、招待先と一致せずアクセスできない。

確認する順番:

1. 招待先メールアドレスに誤りがないか。
2. 相手が同じメールアドレスでBoxにログインしているか。
3. 招待メールが迷惑メールに入っていないか。
4. 招待メールの有効期限が切れていないか。
5. フォルダオーナーまたは共同所有者が招待しているか。

## FAQ 2. 共有リンクを送ったが開けない

共有リンクが開けない場合は、リンク設定を確認する。社外秘以上のフォルダでは、共有リンクが無効化されている場合がある。この場合、共有リンクではなく外部コラボレーター招待を利用する。

確認する項目:

- リンクの有効期限
- パスワード設定
- リンクの対象範囲
- ダウンロード許可
- フォルダの機密区分

## FAQ 3. 取引先ドメイン許可申請が必要な場合

取引先の会社メールドメインが許可リストに登録されていない場合、招待が失敗するか、相手がアクセスできないことがある。一般社員は許可リストを直接変更できない。

申請手順:

1. 部門長に共有目的と対象フォルダを説明し、承認を得る。
2. SharePointの「取引先ドメイン許可申請」フォームを開く。
3. 取引先会社名、メールドメイン、共有目的、共有期間、対象フォルダURLを入力する。
4. 情シス部門が確認し、承認後に許可リストへ追加する。
5. 許可完了後、フォルダオーナーまたは共同所有者が再招待する。

## FAQ 4. すぐに情シスへエスカレーションすべきケース

次の場合は、社員側で判断せず情シスへ問い合わせる。

- 取引先ドメインが未登録である
- フォルダの機密区分が社外秘以上である
- フォルダオーナーが退職済み、または不明である
- 共有先が複数社にまたがる
- エラーメッセージに権限、ポリシー、制限が表示される
- 個人情報、契約書、設計資料を含むフォルダである

## FAQ 5. 回答テンプレート

取引先からアクセス不可の連絡を受けた場合は、次のように案内する。

「招待先メールアドレス、ログイン中のメールアドレス、招待メールの受信状況を確認してください。あわせて、当社側でフォルダの機密区分、共有リンク設定、取引先ドメインの許可状況を確認します。ドメイン未登録または社外秘フォルダの場合は、情シス承認後に再招待します。」
